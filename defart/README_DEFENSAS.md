# Bloque 10 - Defensas

Modulo de defensas conectado a BaseLocal.

Flujo recomendado:
1. Entrar a Requisito.
2. Crear o seleccionar periodo.
3. Cargar y analizar Excel.
4. Entrar a Defensas.
5. Filtrar por periodo, carrera, sede, modo o busqueda.
6. Seleccionar estudiante para generar mensaje individual.

Incluye agenda, resumen por carrera, tribunales/aulas, mensajes, exportacion CSV, JSON e ICS.
