# Bloque 11 - Infor

Modulo informativo conectado a BaseLocal.

Flujo recomendado:
1. Entrar a Requisito.
2. Crear o seleccionar periodo.
3. Cargar y analizar Excel.
4. Entrar a Infor.
5. Revisar resumen, guia rapida o comunicado.

Incluye resumen de titulacion, guia de uso del bloque Requisitos, comunicado copiable y exportacion JSON.
No modifica Titulos.
