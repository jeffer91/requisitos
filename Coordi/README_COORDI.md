# Bloque 8 - Coordi

Panel de coordinacion conectado a BaseLocal.

Flujo recomendado:
1. Entrar a Requisito.
2. Crear o seleccionar periodo.
3. Cargar y analizar Excel.
4. Entrar a Coordi.
5. Revisar prioridad alta, carreras y requisitos criticos.

Incluye resumen por carrera, requisitos con mayor atencion, estudiantes priorizados y mensajes rapidos.
