# BDLocal/legacy

Compatibilidad temporal.

Esta carpeta servirá para guardar puentes mientras se migra BDLocal hacia una estructura más modular.

Regla:

No se deben borrar ni mover archivos antiguos hasta que cada pantalla esté probada.

Objetivo:

- mantener compatibilidad con pantallas actuales
- evitar romper rutas existentes
- permitir migrar Firebase, sincronización y adaptadores por bloques
