# BDLocal/core

Núcleo de BDLocal.

En esta carpeta se moverán después, de forma controlada, los archivos centrales de la base local:

- configuración
- esquema IndexedDB
- claves internas
- apertura de base
- estado
- caché
- arranque

Regla: este módulo no debe depender de Firebase, Supabase, Excel ni Google Sheets.

Estado actual: carpeta preparada. No se mueve ningún archivo todavía para no romper compatibilidad con las pantallas actuales.
