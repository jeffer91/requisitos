# BDLocal/connections/supabase

Conector Supabase.

Rol:

- nube secundaria automática
- respaldo de datos manuales y críticos
- salida segura cuando Firebase falla o queda sin cuota

Datos prioritarios:

- notas
- divisiones manuales
- Telegram
- títulos
- decisiones finales
- historial crítico

Regla:

Supabase no debe duplicar todo el Excel por defecto. Debe proteger principalmente lo que no se puede reconstruir fácilmente.
