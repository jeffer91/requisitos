# BDLocal/data/validators

Validadores de datos antes de guardar en BDLocal.

Función:

- validar estudiante
- validar período
- validar nota
- validar división
- validar Telegram
- validar título
- validar datos manuales críticos

Regla: aquí no se corrige automáticamente. Solo se valida y se informa si un dato está incompleto, incorrecto o riesgoso.
