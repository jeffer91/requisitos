# Bloque 9 - Repor

Modulo de reportes conectado a BaseLocal.

Flujo recomendado:
1. Entrar a Requisito.
2. Crear o seleccionar periodo.
3. Cargar y analizar Excel.
4. Entrar a Repor.
5. Generar reporte general, por carreras, por requisitos o pendientes criticos.

Incluye vista previa, copiar reporte, imprimir, exportar TXT, exportar HTML y exportar JSON.
