# Carga

Nueva carpeta de carga inteligente de datos.

Reemplaza a `Gestion/Excel`.

Funciones iniciales:
- Lectura de CSV, TXT, JSON, XLSX y datos pegados.
- Mapeo de campos antiguos a campos internos.
- Detección de período, carrera, requisitos, notas y división.
- Normalización previa a BDLocal.
- Validación de errores graves antes de guardar.
- Guardado en BDLocal usando repositorios.
